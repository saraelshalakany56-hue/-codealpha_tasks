# CodeAlpha Data Analytics Internship Projects

 Overview

This repository contains the tasks completed during my Data Analytics Internship at CodeAlpha.  
The projects demonstrate practical skills in data collection, cleaning, analysis, visualization, and Natural Language Processing (NLP) using Python and Google Colab.

The workflow of the project follows a complete data analytics pipeline:
1. Data Collection using Web Scraping
2. Exploratory Data Analysis (EDA)
3. Advanced Data Visualization
4. Sentiment Analysis using NLP

# Task 1 — Web Scraping

Objective
To collect book data from a public website using Python web scraping techniques.

 Website Used
https://books.toscrape.com/

 Tools & Libraries
- Python
- Google Colab
- Requests
- BeautifulSoup
- Pandas

 Extracted Data
- Book Title
- Price
- Rating
- Availability

 Output
The scraped data was exported into Excel format for further analysis.


# Task 2 — Exploratory Data Analysis (EDA)

 Objective
To clean, explore, and analyze the scraped dataset using statistical analysis techniques.

 Analysis Performed
- Data cleaning and preprocessing
- Statistical summaries
- Price analysis
- Rating analysis
- Dataset exploration
- Insights extraction

 Tools Used
- Pandas
- Matplotlib

 Key Insights

- The dataset was successfully cleaned and prepared for visualization.
